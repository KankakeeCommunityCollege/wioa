Overview

This section describes how you can add simple text formatting to your job posting. This text formatting allows you to use a shorthand, in some of the form responses, to create bold and italicized text, paragraphs, and bulleted lists.

Important!

Text formatting will only work in your multiline-text responses to the following form questions:

  • Job Description
  • Job Requirements
  • Pay & Benefits

Paragraph text

For paragraphs, create separation from other paragraphs (and from bulleted lists) using blank lines.
Bulleted lists are described later.

Example shorthand
This short sentence becomes the first paragraph.

This sentence becomes a second paragraph of text.

This paragraph introduces the following bulleted list:

- list item 1
- list item 2
Rendered example

This short sentence becomes the first paragraph.

This sentence becomes a second paragraph of text.

This paragraph introduces the following bulleted list:

  • list item 1
  • list item 2

Bold text

Bold (or strong) text can be created by surrounding your text with either double asterisks (**) or double underscores (__).

Example shorthand
**bold text 1** and __bold text 2__
Rendered example

bold text 1 and bold text 2


Italicized text

Italicized text (emphasis) can be created by surrounding your text with asterisks (*) or underscores (_).

Example shorthand
*italicized text 1* and _italicized text 2_
Rendered example

italicized text 1 and italicized text 2


External link

An external link can be created by placing the link-text between square brackets followed by the URL between parenthesis.

Please note: All links will open in a new window or tab.

Example shorthand
Check out this [link text](https://google.com)
Rendered example

Check out this link text


Strikeout text

Strikeout (i.e. stricken, strikethrough) text can be created by surrounding your text with tilde (~).

Example shorthand
~Bartender~ **(no longer available)**
Rendered example

Bartender (no longer available)


Bulleted lists

To create bulleted list items, place each list item on its own line. Prefix each line/list-item withe either a hyphen and space (-) or asterisk and space (*.) Be sure to add a blank line before and after the first and last list items.

Example shorthand
- list item 1
- list item 2

Or using asterisks:

* list item 1
* list item 2
* list item 3
Rendered example
  • list item 1
  • list item 2

Or using asterisks:

  • list item 1
  • list item 2
  • list item 3

Example job requirements formatting

The following example shows a combination of different formatting shorthands.

Example shorthand
**Duties and Essential Job Functions:**

* Provide superior customer service leadership; greet and assist customers.
* Operate cash register and scanner to itemize and total customer's purchase, collect payment from customers and make change, bag merchandise, and assist customers with merchandise as necessary.
* Follow company work processes to receive, open and unpack boxes, cartons and totes of merchandise; stock merchandise, restock and rotate merchandise on shelves, and build merchandise displays.
* Clean the store; take out trash; dust and mop store floors; clean restroom and stockroom; and help set up sidewalk displays.
* Assist in implementation and maintenance of planograms.
* Open and close the store under specific direction of the Store Manager.
* Perform additional duties typically performed by the Store Manager or Assistant Store Manager, in their absence.

***Other important equirements* may include...**

**For full details please visit the [online job posting](https://www.google.com).**
Rendered example

Duties and Essential Job Functions:
